# Backend 로드맵

![초보 백엔드 개발자 로드맵](https://raw.githubusercontent.com/berenickt/image-server/main/img/beginner-Backend-developer-roadmap.png)

- `API Server` : 프론트와 백엔드끼리의 연결을 가능하게 해주는 영역
  - e.g. NestJS / Django / Spring
  - 통신 종류 : REST API / GraphQL / gRPC
- `Database` : 데이터 저장소
  - MySQL
  - PosterSQL
  - Oracle
  - MongoDB
  - DynamoDB
- `Infrastructure` : 백앤드를 실행하는 하드웨어를 관리하는 영역
  - Cloud / K8s / DevOps

## 1. 1~4주차

- 언어 선택
  - 자바스크립트 or 파이썬
  - ❌ 자바 금지
  - ❌ 고랭, 러스트 금지
  - ❌ 타입스크립트 금지
- 선택한 언어의 기초만 배우기
  - 변수
  - 함수
  - 조건문
  - 반복문
- 4가지 기초 개념으로 미니 프로젝트 만들기
  - 가위바위보
  - 구구단 출력기
  - 계산기

---

## 2. 5~8주차

- 개념 배우기
- API, HTTP
- 웹 프레임워크 익숙해지기
  - Node.js
    - Fastify
    - NestJS
  - Python
    - FastAPI
- API 서버 만들기
  - 간단한 게시판(필수)
    - 글 작성, 수정, 삭제, 조회

---

## 3. 9~12주차

- DB 배우기
  - 데이터베이스가 뭔지 배우기
  - 왜 사용하는지 배우기
- 데이터베이스 선택하기
  - PosterSQL
  - MySQL
- SQL 배우기
  - Insert, Select, Update, Delete
- 게시판 API 서버에 DB 붙여보기
  - 게시판 글 데이터를 DB에 넣기
  - 회원가입/로그인 만들기
