# 2. 알고리즘을 잘 공부하는 방법

## 2.1 문제를 풀 때 중요한 것

1. 항상 `여러가지 풀이 방법`이 있을 수 있다는 것을 기억하자
2. 항상 `예외가 있을 수 있다는 것`을 기억하자
3. 내가 풀어낸 답이 `베스트인지 의심`하자
4. 문제를 풀었다면 `시행착오를 모두 기록`하자
5. `다른 사람의 코드`를 많이 보자. 생각하지 못했던 방법을 발견할 수 있다.
6. 쉽게 포기하지 말자. 하지만 `도저히 모르겠다면 답을 보는 것`도 좋은 방법이다.
