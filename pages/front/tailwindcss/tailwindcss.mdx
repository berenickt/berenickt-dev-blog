## 1. Tailwindcss

https://tailwindcss.com/docs/installation

- Utility-first CSS framework,
- 즉 미리 정의된 CSS의 특성들을 `class` 를 통해 불러와 적용시키는 형태의 CSS 프레임워크
- 원하는 `html` 요소에 `class` 을 부여해주면 바로 사용이 가능하다

cf. VSCode 확장기능

- Tailwind CSS IntelliSense : 클래스명 자동완성
- Tailwind Docs : 문서 찾아볼 떄 Ctrl + Shipt + P로 검색

## 2. 예제 1 : 기본 사용법

https://tailwindcss.com/docs/installation

```sh
npm install -D tailwindcss
npx tailwindcss init
```

tailwindcss를 사용하고자 하는 경로, 파일 작성(**확장자명은 띄워쓰면 안됨**)

```js
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: ['./src/**/*.{html,js,jsx,ts,tsx}'],
  theme: {
    extend: {},
  },
  plugins: [],
}
```

전역 CSS에 다음 코드 추가 (CRA에서는 index.css 맨 위에 추가)

```css
/* tailwind 기본 스타일 및 확장된 스타일 주입 */
@tailwind base;

/* tailwind 기본 컴포넌트 클래스 및 확장된 컴포넌트 클래스 주입 */
@tailwind components;

/* tailwind 기본 유틸리티 클래스 및 확장된 유틸리티 클래스 주입 */
@tailwind utilities;
```
